# Screenshot Guide for Chrome Web Store

## Requirements
- **Size**: 1280x800 pixels or 640x400 pixels (16:10 ratio)
- **Format**: PNG or JPEG
- **Minimum**: 1 screenshot required
- **Recommended**: 3-5 screenshots
- **File size**: Each under 5MB

## Tools for Screenshots

### Option 1: Chrome DevTools (Recommended for exact sizing)
1. Open extension
2. Press F12 ‚Üí Console
3. Run: `window.resizeTo(1280, 800)`
4. Use Chrome's built-in screenshot tool
5. Or use Cmd+Shift+4 (Mac) / Windows+Shift+S (Windows)

### Option 2: Online Resize Tool
1. Take screenshots normally
2. Go to: https://www.iloveimg.com/resize-image
3. Resize to exactly 1280x800 pixels
4. Download PNG

### Option 3: Figma/Photoshop
1. Create 1280x800 canvas
2. Paste screenshot
3. Center and crop
4. Export as PNG

## Screenshot Checklist

Before uploading, make sure:
- [ ] All screenshots are 1280x800 pixels
- [ ] File format is PNG (preferred) or JPEG
- [ ] No personal information visible (real names, emails)
- [ ] Extension looks professional and polished
- [ ] Server status shows "Online" (green)
- [ ] Text is readable and clear
- [ ] Colors look good and vibrant
- [ ] No browser warnings or errors visible
- [ ] Extension icon is visible where applicable

## Pro Tips

1. **Use Dark Mode** - Netflix/YouTube screenshots look better in dark mode
2. **Show Pizza Branding** - Include the pizza emoji üçï where visible
3. **Real Usage** - Make it look like actual use, not staged
4. **Clean Chat** - Use fun but appropriate test messages
5. **Good Contrast** - Light UI on dark backgrounds or vice versa
6. **No Clutter** - Close unnecessary browser tabs/windows

## After Taking Screenshots

1. Save them as:
   - `screenshot1_landing.png`
   - `screenshot2_host_room.png`
   - `screenshot3_watching_netflix.png`
   - `screenshot4_chat.png`
   - `screenshot5_guide.png`

2. Store in a `/screenshots` folder

3. Review each one:
   - Is it clear and professional?
   - Does it show a key feature?
   - Is the text readable?

4. Upload to Chrome Web Store in order (1-5)
